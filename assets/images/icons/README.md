# 아이콘 배치 가이드

다음 파일명을 사용해 이미지를 넣어주세요. 해상도는 3배(@3x) 권장, 배경은 투명 PNG를 추천합니다.

필수:
- assets/images/icons/tab-list.png (비활성)
- assets/images/icons/tab-list-active.png (활성)
- assets/images/icons/tab-calendar.png (비활성)
- assets/images/icons/tab-calendar-active.png (활성)
- assets/images/icons/tab-map.png (비활성)
- assets/images/icons/tab-map-active.png (활성)
- assets/images/icons/camera.png (카메라 FAB 아이콘)

선택(있으면 코드 확장 가능):
- assets/images/icons/camera-pressed.png (카메라 FAB 눌림 상태)

사이즈 가이드(권장 렌더 크기):
- 탭 아이콘: 26x26 px
- 카메라 FAB: 26x26 px (FAB 배경 원은 코드에서 적용됨)

스케일 에셋(@2x/@3x) 사용 팁:
- 위 파일명 그대로 넣고, 추가로 동일 이름에 @2x/@3x 접미사를 둔 파일을 같이 넣으면 기기 해상도에 따라 자동 선택됩니다. 예) tab-list.png, tab-list@2x.png, tab-list@3x.png

이미지 교체 후 새로고침 또는 캐시 초기화가 필요할 수 있습니다.
